# Requirements: Copyright Page

## Overview

ISMS Guide サイトにコンテンツの著作権・出典に関する説明を追加する。ISO/JISC への事前問い合わせ結果を根拠として、本サイトのコンテンツが適法に作成されていることを明示する。

## 必須記載事項

1. **ISO・JISC への事前問い合わせ実施済み** であること
2. **ISO から購入した規格原文をベースに、自身で解釈・要約して作成** したオリジナルコンテンツであること
3. **JISC（JIS規格）のコンテンツの流用ではない** こと
4. JISC 回答の根拠:「規格を読み込み、ご自身の言葉で概要や要約等を作成される場合には引用の対象外」

## User Stories

### US-1: サイト閲覧者としてコンテンツの出典・正当性を確認したい

**As** ISMS Guide を参照する組織の担当者
**I want** このサイトのコンテンツがどのように作成されたか知りたい
**So that** 安心してコンテンツを参照・活用できる

#### Acceptance Criteria

1. **AC-1.1**: WHEN ユーザーがトップページを閲覧した時 THE SYSTEM SHALL コンテンツの出典・著作権に関する説明セクションを表示する
2. **AC-1.2**: WHEN ユーザーが説明セクションを閲覧した時 THE SYSTEM SHALL ISO・JISC に事前問い合わせを行ったことを明示する
3. **AC-1.3**: WHEN ユーザーが説明セクションを閲覧した時 THE SYSTEM SHALL ISO から購入した規格原文をベースに自身で解釈・作成したコンテンツであることを明示する
4. **AC-1.4**: WHEN ユーザーが説明セクションを閲覧した時 THE SYSTEM SHALL JISC（JIS規格）コンテンツの流用ではないことを明示する

### US-2: サイト閲覧者として問い合わせの根拠を確認したい

**As** 著作権について詳しく知りたい利用者
**I want** ISO・JISC への問い合わせ内容と回答を確認したい
**So that** 本サイトの正当性を自分で判断できる

#### Acceptance Criteria

1. **AC-2.1**: WHEN ユーザーがトップページの説明を見た時 THE SYSTEM SHALL 詳細ページへのリンクを表示する（または詳細を展開できる）
2. **AC-2.2**: WHEN ユーザーが詳細を確認した時 THE SYSTEM SHALL JISC からの回答（要約作成は引用対象外）を表示する
3. **AC-2.3**: WHEN ユーザーが詳細を確認した時 THE SYSTEM SHALL ISO への問い合わせ経緯を表示する

### US-3: サイト閲覧者として正規規格の入手方法を知りたい

**As** ISO 27001 の原文を入手したい利用者
**I want** 正規の規格入手方法を知りたい
**So that** 必要に応じて原文を参照できる

#### Acceptance Criteria

1. **AC-3.1**: WHEN ユーザーが著作権セクションを閲覧した時 THE SYSTEM SHALL ISO 規格の正式な入手先（ISO Store、JSA Webdesk）へのリンクを表示する

## 配置オプション

### Option A: トップページに直接記載（推奨）
- トップページの下部に「本サイトのコンテンツについて」セクションを追加
- 簡潔に要点を記載し、詳細は折りたたみまたは別ページ

### Option B: 専用ページ + トップページにリンク
- `/about/copyright.md` に詳細ページを作成
- トップページのフッター近くにリンクを配置

## Non-Functional Requirements

### Visibility

- **NFR-1**: トップページから 1 クリック以内でアクセス可能
- **NFR-2**: 目立つ位置に配置（スクロールしなくても見える、または features の直後）

### Credibility

- **NFR-3**: 問い合わせ日時または時期を記載
- **NFR-4**: 問い合わせ先（ISO Central Secretariat、日本規格協会）を明記

### Maintainability

- **NFR-5**: 将来の規格改定・追加問い合わせ時に更新しやすい構造

## Out of Scope

- 問い合わせメール原文の全文掲載（要約で十分）
- プライバシーポリシー
- 利用規約
