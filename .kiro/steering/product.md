# Product Overview

## Purpose

ISO/IEC 27001認証取得を目指す組織向けに、フォーク可能なISMSテンプレートリポジトリと包括的な解説サイトを日本語で無償公開する。

## Background & Problem

ISO 27001の認証取得において、多くの組織が以下の課題に直面している：

- ISMS文書の作成に多大な工数がかかる
- 何を文書化すべきか分かりにくい
- コンサルティング費用が高額
- 日本語の実践的なガイドが少ない

## Solution

| 提供物 | 説明 |
|--------|------|
| 解説サイト | ISO 27001の要求事項・管理策を分かりやすく日本語で解説 |
| テンプレートリポジトリ | フォークしてすぐに使えるISMS文書一式 |
| フォーク利用ガイド | リポジトリのカスタマイズ手順 |

## Target Users

| 優先度 | 対象 | 提供形式 |
|:------:|------|----------|
| 高 | Git/技術に精通した組織（スタートアップ、IT企業等） | 静的サイト + GitHub |
| 低 | Git不慣れな組織 | Notion / Google Workspace / Microsoft Office版（将来対応） |

## Key Features

- 要求事項解説（箇条4-10）
- 管理策解説（Annex A 93項目）
- 用語集
- ISMSマニュアルテンプレート
- 適用宣言書（SoA）テンプレート
- 各種手順書テンプレート
- 記録テンプレート
- 解説とテンプレートの相互リンク（双方向参照でナビゲーションを支援）

## Business Model

無償公開（オープンソース）
