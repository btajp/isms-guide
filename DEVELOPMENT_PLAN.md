## 1. プロジェクト概要

### 1.1 ビジョン

ISO/IEC 27001認証取得を目指す組織向けに、**フォーク可能なISMSテンプレートリポジトリ**と**包括的な解説サイト**を日本語で無償公開する。

### 1.2 ソリューション

| 提供物 | 説明 |
|--------|------|
| **解説サイト** | ISO 27001の要求事項・管理策を分かりやすく日本語で解説 |
| **テンプレートリポジトリ** | フォークしてすぐに使えるISMS文書一式 |
| **フォーク利用ガイド** | リポジトリのカスタマイズ手順 |

### 1.3 ターゲットユーザー

| 優先度 | 対象 | 提供形式 |
|:------:|------|----------|
| **高** | Git/技術に精通した組織（スタートアップ、IT企業等） | 静的サイト + GitHub |
| 低 | Git不慣れな組織 | Notion / Google Workspace / Microsoft Office版（将来対応） |

## 2. 技術スタック

| 項目 | 技術 |
|------|------|
| 静的サイトジェネレータ | **VitePress 1.5.x** |
| ホスティング | **Cloudflare Pages** |
| ドメイン | isms-guide.jp（取得予定） |
| バージョン管理 | GitHub |
| ドキュメント形式 | Markdown |
| 図表 | Mermaid（vitepress-plugin-mermaid） |
| CI/CD | Cloudflare Pages 自動デプロイ |

## 3. サイト構造

```
isms-guide.jp
│
├── /docs              ← 解説コンテンツ
│   ├── /requirements/    要求事項解説（箇条4-10）
│   ├── /controls/        管理策解説（Annex A 93項目）
│   └── /glossary/        用語集
│
├── /isms              ← フォーク対象テンプレート
│   ├── /manual/          ISMSマニュアル
│   ├── /soa/             適用宣言書（SoA）
│   ├── /procedures/      手順書
│   └── /records/         記録テンプレート
│
└── /guide             ← フォーク・カスタマイズ手順
```
