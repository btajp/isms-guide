# ISMS テンプレートディレクトリ

このディレクトリは、組織固有の ISMS 文書を配置するためのテンプレートです。

## ディレクトリ構造

```
isms/
├── manual/         # ISMS マニュアル（基本方針、適用範囲など）
├── soa/            # 適用宣言書（Statement of Applicability）
├── procedures/     # 手順書（運用手順、対応手順など）
└── records/        # 記録（監査記録、インシデント記録など）
```

## 使い方

### 1. リポジトリをフォーク

このリポジトリをフォークして、組織専用のリポジトリを作成します。

### 2. 組織情報の設定

`docs/.vitepress/config.mts` を編集して、組織名やサイト URL を設定します。

### 3. 文書の配置

各ディレクトリに組織の ISMS 文書を配置します：

- **manual/**: ISMS マニュアル、情報セキュリティ方針など
- **soa/**: 適用宣言書（SoA）
- **procedures/**: 各種手順書
- **records/**: 監査記録、レビュー記録など

### 4. プライベート設定

組織の機密文書が含まれる場合は、リポジトリをプライベートに設定してください。

## 注意事項

- このディレクトリの内容は `.gitignore` に追加することを推奨します
- 機密情報を GitHub にプッシュしないよう注意してください
- 文書のバージョン管理には Git の機能を活用してください
