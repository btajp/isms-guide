## 5. 評価基準

### 機密性

| レベル | 説明 |
|:------:|------|
| 3 | 極秘：漏えいすると重大な損害 |
| 2 | 社外秘：漏えいすると相当の損害 |
| 1 | 公開：漏えいしても影響なし |

### 完全性

| レベル | 説明 |
|:------:|------|
| 3 | 改ざんされると重大な損害 |
| 2 | 改ざんされると相当の損害 |
| 1 | 改ざんされても影響が限定的 |

### 可用性

| レベル | 説明 |
|:------:|------|
| 3 | 停止すると重大な損害（数時間で影響） |
| 2 | 停止すると相当の損害（1日で影響） |
| 1 | 停止しても影響が限定的 |

### 資産価値

資産価値 = 機密性 + 完全性 + 可用性（最大9）

- 資産の追加・変更・廃棄時に更新
